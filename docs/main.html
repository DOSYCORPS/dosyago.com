Dosyago is currently building 2 secret products.

Litewait (at litewait.io) is a web browser that saves you data and bandwidth costs, by reducing the amount of CSS, HTML and JavaScript sent to you, while still delivering you the content and functions, of the pages you visit.

We built litewait to save ourselves money when traveling and buying 4G data sims, because sometimes it's hard to unlimited data for a good price. Another bonus is our phone runs less hot (because less code means less battery used) and so we can browser for longer. A downside is you don't get any images, fancy looking design, videos, or other things. And the sites can look a bit different to what you're used to.

Maybe the downsides will mean that people will not find litewait useful. But we think that in the cases where you need access to the web regularly but that access is too expensive, or the connection is to slow, to use a regular browser, litewait is a useful option.
